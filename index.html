CSS:

.suggested{
  color: #fff;
  background-color: #000;
  /* whatever styles you like */
}

JavaScript:

$('#spacetagging-input').spaceTagging('#spacetagging-list a');

Documentation

Syntax:

$(element).spaceTagging(elements);

Options:

  • suggestedClass - (string, defaults to 'suggested') A string indicating the class name to be added when it's tagged.
